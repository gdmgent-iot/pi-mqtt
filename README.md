# pi-mqtt 

## install mosquitto

```
sudo apt install mosquitto
```

## make and activate venv


```
python -m venv venv
*# For macOS/Linux
source venv/bin/activate
*# For Windows
.\venv\Scripts\activate
```

## install paho-client


```
pip install paho-mqtt
```
